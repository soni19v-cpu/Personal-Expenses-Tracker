# Personal Expense Tracker

A Python command-line application to track your daily income and expenses. The program stores all transactions in a CSV file and provides a detailed summary.

## Features
- Add income and expenses
- Store all records in `expenses.csv`
- View all transactions in tabular format
- Get financial summary:
  - Total Income
  - Total Expenses
  - Net Savings
  - Category-wise Expense Breakdown
- Automatically initializes necessary files

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/Personal-Expense-Tracker
2. Run the script:
   python "Personal Expenses.py"

## Technologies Used
- Python 3
- CSV file handling
- Date & time module

## Purpose
This project helps beginners understand file handling, loops, functions, and CLI-based applications.
